## Installation
For YCBarcodeReader, use the following entry in your Podfile:

```rb

pod 'YCBarcodeReader'

```

Then run `pod install`.

In any file you'd like to use YCBarcodeReader in, don't forget to import the framework with import YCBarcodeReader.
